## 🎯 新增功能

### 1. 完善标点符号删除

**问题**：
- 部分中文标点符号没有被删除（如 `……`、`【`、`】` 等）

**解决方案**：
- 扩充 `PUNCTUATION` 数组
- 新增标点符号：
  - `【`、`】`（方括号）
  - `「`、`」`（单引号）
  - `『`、`』`（双引号）
  - `《`、`》`（书名号）
  - `…`、`……`（省略号）

### 2. 优化时长显示格式

**问题**：
- 时长格式显示为 "X.XX秒"（小数格式）
- 用户期望格式为 "XX分钟XX秒" 或 "XX秒"（整数格式）

**解决方案**：
- 修改时长计算逻辑
- 新的格式规则：
  - < 1秒：显示毫秒数（如 "6毫秒"）
  - 1-59秒：显示秒数（如 "4秒"）
  - ≥ 60秒：显示分钟和秒数（如 "1分30秒"）

## 🔧 技术实现

### 后端修改

**1. 扩充 PUNCTUATION 数组**：
```typescript
// 修改前
const PUNCTUATION = ['，', '。', '！', '？', '；', '：', ',', '.', '!', '?', ';', ':', '、', '（', '）', '(', ')'];

// 修改后
const PUNCTUATION = ['，', '。', '！', '？', '；', '：', ',', '.', '!', '?', ';', ':', '、', '（', '）', '(', ')', '【', '】', '「', '」', '『', '』', '《', '》', '…', '……'];
```

**2. 优化时长计算逻辑**：
```typescript
// 修改前
const durationText = duration < 1000 ? `${duration}毫秒` : `${(duration / 1000).toFixed(2)}秒`;

// 修改后
let durationText = '';
if (duration < 1000) {
  durationText = `${duration}毫秒`;
} else if (duration < 60000) {
  const seconds = Math.floor(duration / 1000);
  durationText = `${seconds}秒`;
} else {
  const minutes = Math.floor(duration / 60000);
  const seconds = Math.floor((duration % 60000) / 1000);
  durationText = `${minutes}分钟${seconds}秒`;
}
```

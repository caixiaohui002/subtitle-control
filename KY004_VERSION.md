## 🎯 新增功能

### 处理时长显示

**功能说明**：
- 在处理结果框标题旁边显示处理时长
- 使用灰色字体，不醒目
- 格式：
  - 小于1秒：显示为"毫秒"
  - 大于1秒：显示为"秒"（保留2位小数）

**显示位置**：
- 位置：处理结果框标题旁边
- 样式：`text-xs text-gray-400`（小号灰色字体）
- 显示内容：`耗时 X.XX秒` 或 `耗时 XXX毫秒`

## 🔧 技术实现

### 后端修改

**1. 记录开始时间**：
```typescript
const startTime = Date.now();
```

**2. 计算处理时长**：
```typescript
const endTime = Date.now();
const duration = endTime - startTime;
const durationText = duration < 1000 ? `${duration}毫秒` : `${(duration / 1000).toFixed(2)}秒`;
```

**3. 发送结果时包含时长**：
```typescript
controller.enqueue(encoder.encode(
  `data: ${JSON.stringify({ type: 'result', content: result, duration: durationText })}\n\n`
));
```

**4. 调试日志**：
```typescript
sendDebug(`处理完成！耗时：${durationText}`);
```

### 前端修改

**1. 添加状态**：
```typescript
const [processingDuration, setProcessingDuration] = useState('');
```

**2. 接收时长信息**：
```typescript
if (data.type === 'result') {
  result = data.content;
  setOutputText(result);
  setProcessingDuration(data.duration || '');
}
```

**3. 重置时长**：
```typescript
setProcessingDuration('');
```

**4. 显示时长**：
```typescript
{processingDuration && (
  <span className="text-xs text-gray-400">
    耗时 {processingDuration}
  </span>
)}
```
